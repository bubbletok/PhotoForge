# [게임소개]

여러 장의 사진을 활용하여 기믹을 수행하고 탈출하는 퍼즐 어드벤쳐 게임.

사진을 이동시켜서 길을 만들거나, 겹쳤을 때 활용할 수 있는 특수한 기믹을 통해 스테이지를 클리어 하는 것이 목적이다.

### 조작 방법

W,A,S,D - 플레이어 이동

마우스 클릭 & 드래그 - 사진 움직이기
